Goalie Tracker Pro v1.5

기준 버전:
- v1.1을 기준으로 수정했습니다.
- v1.2와 v1.3의 단순화 변경은 포함하지 않았습니다.

일반 경기:
- 우리 골리 기록은 v1.1과 동일한 +/- 방식 유지
- 각 피리어드에 상대 골리 기록 추가
- 상대 골리 SOG = 우리 팀 유효슈팅
- 상대 골리 GA = 우리 팀 득점
- PC/태블릿에서는 좌우 비교
- 좁은 스마트폰에서는 위아래 비교
- 양쪽 SAVE% 자동 계산

슛아웃:
- 우리 골리는 기존 상세 기록 그대로 유지
  막음/실점, 슈팅 방향, 슈팅 종류, 메모
- 상대 골리는 결과만 간단히 기록
  상대 막음 / 우리 득점
- 양쪽 막음, 실점, SAVE% 자동 비교

기존 데이터:
- 기존 경기와 호환
- 기존 우리 골리 기록은 유지
- 새 상대 골리 기록은 0부터 시작

GitHub 적용:
1. 압축 해제
2. 저장소 Add file → Upload files
3. 파일 전부 업로드
4. Commit changes
5. 1~3분 후 온라인 상태에서 앱 새로고침
6. 설치 앱을 완전히 종료한 뒤 다시 실행
