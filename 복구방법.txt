GOALIE TRACKER PRO - FINAL RESTORE

이 ZIP은 지금까지 완성한 최종 복구본입니다.
기준: v1.5 전체 기능 + v1.6 작은 버튼 패치

복구되는 기능
- 골리 관리
- 경기 생성·저장·수정
- 우리 골리 피리어드별 SOG / GA / SAVE%
- 상대 골리 피리어드별 SOG / GA / SAVE% 비교
- 우리 골리 실점 상세 분석
- 우리 골리 슛아웃 상세 기록
- 상대 골리 슛아웃 결과 비교
- 시즌 통계
- JSON 백업·복원
- PWA 설치 및 오프라인 실행
- 비교 화면 작은 +/- 버튼

GitHub 적용
1. ZIP 압축을 풉니다.
2. goalie-tracker 저장소 첫 화면에서 Add file → Upload files를 누릅니다.
3. 아래 8개 파일을 모두 업로드합니다.
4. 파일이 폴더 안이 아니라 저장소 맨 위에 올라가야 합니다.
5. Commit changes를 누릅니다.
6. 1~3분 후 GitHub Pages 주소를 엽니다.
7. 구버전 화면이면 시크릿 창에서 먼저 확인합니다.
