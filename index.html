📲 홈 화면에 설치

아래 버튼이 보이면 누르세요. 설치 후에는 브라우저 주소창 없이 일반 앱처럼 실행됩니다.

수동 설치 방법
  1. Chrome 오른쪽 위의 ⋮ 메뉴를 누릅니다.
  2. 홈 화면에 추가 또는 앱 설치를 누릅니다.
  3. 설치를 확인합니다.

✈ 오프라인 테스트

  1. 앱 화면을 한 번 정상적으로 엽니다.
  2. 홈 화면 설치를 완료합니다.
  3. 태블릿의 Wi-Fi를 끕니다.
  4. 홈 화면의 Goalie Tracker 아이콘을 누릅니다.
  5. 기록과 저장이 정상 작동하는지 확인합니다.

중요

PWA는 최초 1회 설치할 때 웹 주소로 열어야 합니다. 설치가 끝난 뒤에는 인터넷 없이 사용할 수 있습니다.
